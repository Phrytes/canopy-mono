Nieuwe groep starten

Een groep starten is geen technische ingreep. Het is een afspraak tussen leden. Beantwoord de zes vragen hieronder; ze worden samen het reglement van de groep, zichtbaar voor iedereen die later lid wordt.

Hoe wordt deze groep "vers" gehouden?

De groep gebruikt elke X dagen een nieuwe code. Wie de code niet meer krijgt, valt automatisch uit de groep. De code wissel je buiten de app om uit — bv. via WhatsApp, in een briefje, of in persoon. Dat houdt de groep lokaal.

7. Waar staat de groepsstaat?

Standaard staat de groep op jullie eigen apparaten (geen Solid-pod nodig). Je kunt later upgraden naar een gedeelde pod voor betere persistentie. Eenmaal pod-having → niet meer terug.
