Instellingen

Stoop bewaart je instellingen op twee plekken: dingen die op dit apparaat horen (zoals hoe vaak je telefoon ververst) blijven lokaal; mijn voorkeuren reizen via je pod mee naar al je apparaten.

Op dit apparaat apparaat

Deze instellingen horen bij dít apparaat. Op je telefoon en je laptop kunnen ze verschillen.

Hoe vaak het prikbord ververst Lager = sneller nieuwe posts zien; hoger = zuiniger met data en accu.
Hop-relay (globaal) Mag mijn apparaat berichten doorrelay'en voor andere mensen? Standaard uit: dan doet niemand dat via jou. Per-contact-instelling staat op de contacten-pagina.

Online-window (mobiel)

Op telefoon (V3): hoe vaak en hoe lang mag mijn agent zich tijdelijk verbinden met de relay? Op desktop genegeerd — daar is de agent altijd online.

Window-frequentie Eens per X minuten verbinden. Leeg = altijd verbonden (V2 desktop default).
Window-duur Aantal seconden online per window.

Mijn voorkeuren synct met al je apparaten

Deze voorkeuren horen bij jou — niet bij dit apparaat. Als je inlogt op je pod, reizen ze automatisch mee naar je andere apparaten.

Auto-aanbod-match Mag een onbekend contact mij een aanbod-suggestie sturen die door mijn agent gefilterd wordt? Bij een match krijg je een melding; geen match = stilte.
Standaard locatie delen met nieuwe contacten? Bepaalt of shareLocation default aan staat als je iemand toevoegt. Per-contact altijd handmatig aan/uit te zetten.

Privacy & identiteit

Lokale beheers-acties: gedempte contacten weer aanzetten, en je netwerk-adres roteren (handig na een lek of als je even rust wilt van een specifieke contactketen).

Gedempte contacten Posts en chat van deze peers worden lokaal verborgen. Demping is alleen op dit account zichtbaar — de peers weten het niet.
    Geen gedempte contacten.
    Network-adres rotatie Vervangt je publieke sleutel door een nieuwe. Je stableId blijft hetzelfde — contacten en mute-lijsten volgen je. 7-dagen grace-periode voor in-flight berichten.
